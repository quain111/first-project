About HTML grammar

  1. heading : 'h1'을 사용하며, 끝 맺음에 '/h1' 사용한다.
    'h1'~'h6'까지 있으며, 점점 헤딩 사이즈가 작아진다.

  2. strong : 'strong'을 사용하여 강조를 시켜주며, 끝 맺음이 존재한다.

  3. underline : 'u'를 사용하며, 끝 맺음이 존재한다.

  4. breaking line(줄바꿈) : 'br'를 사용하며, 끝 맺음이 없다.

  5. paragraph : 'p'를 사용하며, 끝 맺음이 있어, 문단의 구분을 해준다.

  6. image : 'img'를 사용하며, "src="picture.jpg""와 같은 심화문법 사용.
    이때, "width="100%""와 같이 이미지의 너비도 조정이 가능하다.

  7. parents and child tag : 'li'로 목록을 작성하고, 'ul'로 그룹화,
    'ol'로 자동 번호 매기기 그룹화가 가능하며, 이때 모든 항목에 'li'처리를 해주어야
    하는데, ctrl키를 누른 상태로 각 줄에 커서 놓을 위치를 클릭해주면 된다.

  8. title : 'title'을 사용하여 해당 페이지의 제목을 알려준다.

  9. meta : '(meta charset='utf-8')'과 같게 코딩한 배경을 토대로 맞추어준다.

  10. head / body : 본문은 'body'로, 글의 내용을 알려주는 상단 요소는 'head'로 표현한다. 끝 맺음 존재한다.

  11. html : 최상위 태그로, 전체 글을 감싸준다. 상단에 (!doctype html)을 표기해주면서 본 페이지가 HTML기반의 페이지임을 관용적으로 나타낸다.

  12. link : 원하는 파트에 링크를 걸어준다. 'a'를 사용하고 끝맺음이 존재한다
    추가적으로, 'a href='주소', target='_blank'로 새 창 띄우기, title='커서 대면 작은 글 상자''등의 표현도 가능하다.

  13. coloring : 컬러링을 할 때, 'style'태그를 사용한 뒤 'color'를 명시해준다.

  14. font: 컬러링과 동일하게 'style' 태그를 사용한다.

  15. highlighting : 'mark' 태그를 사용하여 강조한다.
